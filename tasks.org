* DONE how do we tune logging to show debug level and above?
  CLOSED: [2026-03-14 Sat 18:40]
  Telemere logging can be tuned programmatically using `(tel/set-min-level! :debug)`
  or by setting the JVM system property `taoensso.telemere.min-level=debug`.
  You can also configure specific handlers to filter levels independently.
